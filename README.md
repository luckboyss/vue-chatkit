# vue-chatkit

## 用Vue实现一个实时聊天室
## 实现的主要功能
### 1 拥有多个频道和房间
### 2 显示房间成员和登录状态
### 3 当其他用户输入时进行正在输入提示

## 主要技术栈
### Vue全家桶, bootstrap
### 后端部分使用pusher/chatkit-client帮助我们创建用户、房间等，使我们可以专注前端开发
